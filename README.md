# 🧮 Calculator App

A simple full-stack calculator built using **HTML, CSS, JavaScript, and Node.js (Express)**.

## ✨ Features

* ➕ Add, ➖ Subtract, ✖️ Multiply, ➗ Divide
* ⚠️ Input validation (invalid numbers)
* ❌ Divide-by-zero handling
* 🌙 Dark / ☀️ Light mode toggle
* 🎨 Clean modern UI

## 🚀 Run Locally

```bash
npm install
node index.js
```

Then open:

```
http://localhost:3000
```
